# DoSNet

**Programa de bash para hacer un ataque de denegación de servicio a una IP dada en una red privada**

###### Comandos

- -i  
- -I

**Uso del programa**

```
./DoSNet -i Interfaz de red(eth0,ens33...) -I ip victima  
```
